# Web Apprickation

React project that uses the [Rick and Morty API](https://rickandmortyapi.com/documentation) to show and filter characters from the show.

# Usage

```
# Install dependencies
npm install
```

```
# Run app
npm start
```

```
# Build app
npm run build
```
